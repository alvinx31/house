### 介绍
每天自动获取深圳的新房二手房的成交量
深圳市房地产信息系统：http://ris.szpl.gov.cn/ ([snapshot])

每天的数据写入数据库

![写入数据库](resource/pic1.PNG "Hello")

推荐使用[DB Browser for SQLite](http://sqlitebrowser.org/)查看或者导出数据，支持所有平台。
